This is a great way to get started.

WARNING: You might not be able to scrape any page at any time.

The GET Functionality

To recieve data from crawled websites offered by the api, you must use the following link:
'https://brainsfeedapi-rjsode.herokuapp.com/get?url=[]&key=[]', for getting data in json form
'https://brainsfeedapi-rjsode.herokuapp.com/push?url=[]&key=[]', to push scrawled data to a dataset stored locally

the URL argument

The url argument in the api call should contain the particular URL you want to
scrape for data. for example, if you want to scrape www.heroku.com, then the url argument
must contain the FULL URL , i.e. 'http://www.heroku.com/' in order to scrape sucessfully

https://brainsfeedapi-rjsode.herokuapp.com/get?url=[full-url]

the KEY argument

The KEY argument in the api call is to create authentication implementations on a
fundamental level. The Key for you would be given inside this doc at the bottom. If you
enter the right key, you will be able to access the GET and PUSH Functionality of the API
THe KEY FOR YOU TO ACCESS THE API IS: 'brainsfeed1234'

https://brainsfeedapi-rjsode.herokuapp.com/[get|push]?url=[full-url]&key=[the-key-given('brainsfeed1234')]

The PUSH Functionality

To recieve data from crawlers and directly push it into the .csv file available locally
you must use the urlhttps://brainsfeedapi-rjsode.herokuapp.com/push followed by the url and key arguments as
posted above.
